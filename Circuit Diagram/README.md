# 🔌 Circuit Diagrams

This folder contains the circuit diagrams for the **Smart Solar Panel Monitoring and Cooling Automation** project.

## Contents

* **Circuit Diagram** – Illustrates the complete electrical connections between all hardware components, including:

  * Arduino Uno
  * DHT11 Temperature & Humidity Sensor
  * Dual LDR Sensors
  * SG90 Servo Motor
  * L298N Motor Driver
  * DC Cooling Fan
  * Solar Panel
  * Power Supply Connections

## Purpose

The circuit diagrams provide a detailed reference for assembling the hardware prototype and understanding the interconnections between individual components. They are intended to assist in system implementation, debugging, and future modifications.

## Note

The Arduino source code, project documentation, and hardware images are available in their respective folders within this repository.
